1
Introduction

1.1 Purpose

This user manual provides detailed instructions and guidance to users on how to effectively navigate and utilize the application's features and functionality. It includes a step-by-step procedure for system access and use. It serves as a reference tool, helping users understand how to interact with the application and perform various tasks to achieve their desired outcomes.

1.2 Scope

The user manual covers all essential features, functions, and tasks users can perform on the application. It provides detailed instructions on user registration for user login, viewing and managing customers, products, orders, transactions, total sales, performing backups, and locating backup data. The user manual aims to equip users with the necessary knowledge and guidance to effectively utilize the application's features and enhance their overall user experience.

1.3 Audience

This manual is designed for new and existing users of the application who require guidance on utilizing its features and functionalities. It caters to users with varying technical proficiency and familiarity with the application.

Getting Started

2.1 Accessing the Website

To access the application, users need to launch the application on their device and navigate to the login screen. They can do this by clicking on the application icon and entering their login credentials, such as username and password. Once logged in, users will have full access to the application's features and content directly within the interface.

Navigation

3.1 Main Menu

The main menu serves as a centralized navigation hub, providing users with quick access to various sections and features. Users can find options and shortcuts to essential functions, such as customers, products, orders list, transactions and total sales, enabling seamless navigation and efficiently utilizing the application's capabilities. In addition, features like Backup Data and Locate Backup was included in the main menu.

3.2 Submenus

The submenu provides a secondary level of navigation within specific sections or features. It presents additional options and actions related to the selected menu item, allowing users to efficiently refine their actions, access specific functionalities, and perform context-specific tasks. This includes managing customers, orders, and transactions.

3.3 Search Function

The search function in the application allows users to quickly locate specific information or entities within the application's database. By entering the customer id can initiate a search and retrieve results that match their search query, facilitating efficient data retrieval and navigation within the application.

User Login

4.1 Logging In

To log in to the application, users can navigate to the login screen and enter their registered username and password. Once logged in successfully, users will gain access to their personalized account and be able to utilize the full functionality of the application.

Login to the application using the registered username and password.

5
Customer Management

5.1 Viewing Customer Details

To view the customer list, users can access the customer list section and select the "Customer List" option. This will display a comprehensive list of all customers, including their customer id, name, contact number, address, city and postal code.

5.2 Searching for Customers

To search for customers, users can navigate to the manage customer section and locate the search function. They can enter the customer id to search for a particular customer. The application will then provide search results, allowing users to view customer details, such as customer id, name, contact number, address, city and postal code.

Search customer information using their customer ID.


Search result will display customer information.

5.3 Adding a Customer

The user can add customer information in the manage customer section by clicking the "Add Record" button. Click on it to enter the required customer information, such as customer id, name, contact number, address, city and postal code. Save the information to create a new customer information.

Enter the customer information and click the "Add Record" button.


The application will confirm the added record.

5.4 Updating Customer Information

To update customer information, users can search for a specific customer id by using the search functionality within the manage customer section. They can enter the customer id to search for a particular customer. Once found, users can modify the customer's information, such as name, contact number, address, city and postal code. After making the necessary changes, users can save the updates by clicking the "Update Record."

Edit the data and click "Update Record" to save.


The application will confirm the updated record.

5.5 Deleting a Customer

To delete a customer from the list, users can search for the customer by entering their customer ID within the manage customer section. Once the customer is located, users can permanently remove the customer from the list by clicking the "Delete Record" button.

Search the record you want to delete and click "Delete Record".


The application will confirm if you want to proceed in deleting the record.

5.6 Print Customer List

To print the customer list, users can navigate to the customer list section and select the "Print" button. This will generate a report containing the customer list, which will be saved as an Excel (.xls) file format.

Click the "Print" button to download the list.


The application will confirm the filename of the downloaded file.

Backup Data

10.1 Performing a Manual Backup

To create a data backup in the application, users can access the backup data feature within the homepage. By selecting the "Backup Database," users can initiate the backup process, saving a copy of the application's data, ensuring data integrity and safeguarding against potential data loss or system failures. The backup data will be saved under the user's desktop folder.

Click the "Backup Database" button to backup the files.

10.2 Locate Backup Data

To locate the backup data in the application, users can navigate to the homepage. Within this section, users can access the "Locate Data" button, which provides the path or directory where the backup files are stored. Users can refer to this information to easily locate and access the backup data files for restoration or archival purposes.
